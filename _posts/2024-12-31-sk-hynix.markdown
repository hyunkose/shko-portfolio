## 1. 프로젝트 개요

- **프로젝트명**: SK하이닉스 BI 시스템 구축  
- **수행기관**: (주)디포커스  
- **수행기간**: 2023.10 ~ 2024.12  

본 프로젝트는 SK하이닉스의 반도체 사업 데이터를 기반으로 **재료 원가·매출·판매량 실적과 연·월 단위 판매 목표를 비교 분석할 수 있는 BI 환경 구축**을 목표로 진행되었습니다.

저는 본 프로젝트에서 **DW 기반 데이터 설계 및 BI 전용 데이터소스 구축을 담당**하여, 지표가 **일관된 기준과 신뢰성 있는 구조**로 BI 대시보드에 제공되도록 하는 역할을 수행하였습니다.

---

## 2. 담당 역할 요약 (Data Architecture / DBA 관점)

- DW 내 원천 데이터를 분석하여 **지표 중심 데이터 구조 설계**
- 실적 데이터와 목표 데이터를 비교 가능하도록 **데이터 기준 정렬 및 통합**
- SQL 산출 로직을 기반으로 **Tableau Hyper Extract(Data Mart) 설계·구축**
- 대용량 데이터 환경을 고려한 **성능 및 운영 안정성 확보**

## 4. 주요 성과 및 기여

### 4.1 데이터 구조 설계
- Source 데이터의 기준과 산출 기준을 정렬하여 데이터 일관성 확보
- Fact/Dimension 분리를 통한 효율적인 데이터 관리 구조 구축

### 4.2 SQL 기반 Data Mart 구축
- 복잡한 지표 산출 로직을 SQL로 체계화
- Tableau Hyper Extract를 활용하여 쿼리 성능 최적화

### 4.3 운영 안정성 확보
- 데이터 정합성 검증 프로세스 구축
- 대용량 데이터 환경에서의 안정적 데이터 제공
